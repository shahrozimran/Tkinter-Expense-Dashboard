# TrackWise - Expense Tracker
TrackWise is a robust, Python-based desktop application designed to help users manage their personal finances effectively. Built with a modern Tkinter GUI, it allows for easy tracking, visualization, and management of daily expenses with secure data handling.

## Key Features
Secure Login System: Protects user data with SHA-256 password hashing.

Expense Management: Easily add, update, and delete daily expenses with support for categories and notes.

### Data Visualization:

Pie Charts: Visualize spending distribution by category.

Monthly Bar Charts: Track total expenses month-over-month.

Advanced Filtering: Filter records by specific categories or custom date ranges.

Calendar Integration: Interactive calendar view to browse and add expenses by specific dates.

### Data Portability:

Export: Save your expense reports to CSV.

Import: Bulk load existing data from CSV files.

Recurring Alerts: Set alerts to ensure you never forget to log recurring monthly bills.

## Built With
Python 3

Tkinter (GUI Framework)

Matplotlib (Data Visualization)

tkcalendar (Date Selection)

NumPy (Data Calculation)

CSV (Data Persistence)
